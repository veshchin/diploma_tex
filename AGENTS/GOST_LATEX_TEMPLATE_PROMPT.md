# Промпт для генерации LaTeX-шаблона ВКР (ГОСТ)

Ниже сохранён пользовательский промпт, который фиксирует требования к LaTeX-шаблону
выпускной квалификационной работы (ВКР) по ГОСТ и лучшим практикам.

## Промпт

Действуй как профессиональный LaTeX-разработчик.
Создай полный, модульный и готовый к компиляции шаблон для написания дипломной работы (ВКР),
который строго соответствует государственным стандартам (ГОСТ 7.32-2017, ГОСТ Р 2.105-2019,
ГОСТ Р 7.0.100-2018) и передовым практикам оформления научных текстов.

### Общие требования

- Текст работы должен быть отпечатан на одной стороне листа формата А4 с книжной ориентацией
  (альбомная допускается только для больших таблиц и рисунков).
- Цвет шрифта: чёрный.
- Поля страницы: левое 30 мм, правое 15 мм, верхнее и нижнее 20 мм.
- Шрифт: Times New Roman, размер 14 пт, для сносок используется размер 12 пт.
- Межстрочный интервал: 1.5.
- Абзацный отступ: 1.25 см.
- Страницы нумеруются арабскими цифрами сквозной нумерацией по всему тексту.
  Номер проставляется в центре нижней части страницы без точки.

Лучшие практики LaTeX:

- Для компиляции используй движок `xelatex` или `lualatex`, так как они позволяют подключить
  системный шрифт Times New Roman с помощью пакета `fontspec`.
- Примени модульную структуру: основной файл содержит преамбулу с настройками и команды
  `\input{chapters/...}` для подключения глав.
- Строгое правило вёрстки: в исходных `.tex` файлах каждое предложение должно начинаться
  с новой строки (упрощает поиск ошибок и Git diff).
- Минимизируй ручное форматирование.
  Для таблиц используй `tabularray`, для подписей `caption`.
  Для заголовков и оглавления используй `titlesec` + `titletoc`.

### Титульный лист

- Первая страница ВКР.
- Включается в сквозную нумерацию страниц, но номер не проставляется.
- Реализация: окружение `titlepage` или `\thispagestyle{empty}`.

### Задание

- Оформляется по установленному образцу.
- Не включается в общую нумерацию страниц.
- Реализация: `pdfpages` + `\includepdf{...}`.

### Отзыв

- Отзыв(ы) и рецензия прилагаются к работе, но не включаются в общую нумерацию страниц.
- Реализация: `pdfpages` + `\includepdf{...}`.

### Реферат

- Включается в общую нумерацию страниц, заголовок центрируется.
- 1 абзац: объём работы (страницы), число иллюстраций, таблиц, источников, приложений.
- 2 абзац: ключевые слова (5--15), через запятую, прописными буквами.
- 3 абзац: объект, цель, методы, результаты; объём от 7 строк до конца страницы.
- Реализация: `\chapter*{РЕФЕРАТ}` + `\addcontentsline`.

### Содержание

- Включается в общую нумерацию.
- Заголовок «СОДЕРЖАНИЕ» центрируется, пишется прописными буквами без точки.
- Включает: введение, все разделы/подразделы, заключение, список источников, приложения.
- Подразделы с абзацным отступом относительно разделов.
- Переносы слов в заголовках не допускаются.
- Реализация: `\tableofcontents` + настройка через `titletoc`.

### Введение

- Включается в общую нумерацию.
- Содержит: актуальность, значимость, цель, задачи, объект, предмет, методы, базу.
- Реализация: `\chapter*{ВВЕДЕНИЕ}` + `\addcontentsline`.

### Основная часть

- Включается в общую нумерацию страниц.
- Слово «ОСНОВНАЯ ЧАСТЬ» на отдельном листе по центру.
- Каждый раздел начинается с новой страницы.
- Заголовки разделов: прописными; подразделов: с первой прописной; по центру; без точки.
- Разделы/подразделы нумеруются арабскими (1.1, 1.2), без точки в конце номера.
- Таблицы/формулы/рисунки нумеруются в пределах раздела.
- Реализация: `\chapter{}`, `\section{}`, `\subsection{}` + `titlesec`, счётчики через `chngcntr`/настройки `caption`.

### Заключение

- Включается в общую нумерацию.
- Итоги, выводы, перспективы.
- Реализация: `\chapter*{ЗАКЛЮЧЕНИЕ}` + `\addcontentsline`.

### Список источников

- Включается в общую нумерацию.
- Минимум 10 источников (для ВКР).
- Оформление по ГОСТ Р 7.0.100-2018.
- Ссылки в тексте: арабские цифры в квадратных скобках.
- Рекомендуется `biblatex-gost`.
- Заголовок: «СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ».

### Приложения

- Включаются в общую нумерацию; каждое приложение с новой страницы.
- «ПРИЛОЖЕНИЕ» + буква кириллицы (А, Б, В... исключая Ё, З, Й, О, Ч, Ъ, Ы, Ь).
- Реализация: пакет `appendix` + русская нумерация (например, `\Asbuk{chapter}`) и настройка заголовков через `titlesec`.
