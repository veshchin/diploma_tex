# Промпт для LaTeX-правок (ГОСТ + ограничения проекта)

Этот файл — сохранённый промпт, который помогает генерировать/править LaTeX-шаблон
под требования ГОСТ и правила конкретно этого репозитория.

Каноничные правила проекта — в `AGENTS/AGENTS.md`, карта файлов — в `AGENTS/PROJECT_SUMMARY.md`.

## Промпт (актуальная версия для проекта)

Действуй как профессиональный LaTeX-разработчик.
Правь проект ВКР в репозитории со следующими ограничениями:

1) Не меняй структуру документа и порядок глав без явного подтверждения пользователя.
2) Не добавляй отдельный лист «ОСНОВНАЯ ЧАСТЬ» (в этом проекте он запрещён, даже если встречается в методичках).
3) Сохраняй модульность: точка входа `source/main.tex`, настройки в `source/preamble.tex`, главы в `source/chapters/`.
4) Собирай только `xelatex` или `lualatex` (Times New Roman через `fontspec`).
5) Держи каждый редактируемый `.tex` в пределах ~250–300 строк:
   большие главы режь на `source/chapters/*_partNN.tex`, а файл главы делай тонкой обёрткой с `\chapter{...}` и `\input{...}`.

### Общие требования оформления

- Формат: A4, книжная ориентация (альбомная — только для больших таблиц/рисунков).
- Цвет шрифта: чёрный.
- Поля: левое 30 мм, правое 15 мм, верхнее и нижнее 20 мм.
- Шрифт: Times New Roman 14 пт (для сносок 12 пт, если они используются).
- Межстрочный интервал: 1.5.
- Абзацный отступ: 1.25 см.
- Нумерация страниц: арабская, сквозная; номер внизу по центру, без точки.

### Лучшая практика LaTeX в этом проекте

- Минимизируй ручное форматирование.
- Для таблиц использовать `tabularray`, подписи — `caption`.
- Заголовки и оглавление — через `titlesec` + `titletoc`.
- Абзацы по возможности начинать с новой смысловой строки (полезно для diff).

### Структурные блоки документа

- Титульный лист: первая страница, номер не печатается, но входит в сквозную нумерацию.
- Задание/отзывы/рецензия: подключаются как PDF через `pdfpages`, не входят в общую нумерацию (см. текущую реализацию в `source/main.tex`).
- Реферат: `\chapter*{РЕФЕРАТ}` + `\addcontentsline`.
- Содержание: `\tableofcontents` с заголовком «СОДЕРЖАНИЕ».
- Введение: `\chapter*{ВВЕДЕНИЕ}` + `\addcontentsline`.
- Основная часть: набор нумерованных глав/разделов без дополнительного листа «ОСНОВНАЯ ЧАСТЬ».
- Заключение: `\chapter*{ЗАКЛЮЧЕНИЕ}` + `\addcontentsline`.
- Список источников: `biblatex-gost` (`style=gost-numeric`), заголовок «СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ».
- Приложения: каждое с новой страницы, см. `source/chapters/10_appendices.tex`.
