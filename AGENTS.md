# AGENTS.md

Каноничные инструкции и промты проекта теперь хранятся в директории `AGENTS/`.

- `AGENTS/AGENTS.md` - рабочие правила проекта, текущая карта путей и ограничения по вёрстке.
- `AGENTS/BUILD_AND_OUTPUTS.md` - схема `source/` + `data/` + `results/` + `/.build/` и правила сборки.
- `AGENTS/GOST_LATEX_TEMPLATE_PROMPT.md` - сохранённый промпт по ГОСТ-оформлению LaTeX-шаблона.
- `AGENTS/ПРАВИЛА ВКР.md` - внешние правила и выдержки из методических указаний.
- `AGENTS/СПИСОК ЛИТЕРАТУРЫ.md` - отдельный конспект по библиографии.
- `AGENTS/LESSONS_LEARNED.md` - повторяющиеся ошибки и канонические способы их исправления.

Текущая точка входа проекта: `source/main.tex`.
